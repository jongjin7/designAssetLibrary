# NOVA PWA 구현 및 미해결 이슈 요약

NOVA 앱의 PWA(Progressive Web App) 설치 환경을 구축하면서 확인된 구현 사항과 현재 기술적으로 해결이 어려운 이슈들에 대한 기록입니다.

## 1. 구현 완료 사항
- **PWA 기본 설정**: `manifest.json`, 서비스 워커(`sw.js`), 앱 아이콘(192, 512) 배치 및 수동 설치 유도 로직 구현.
- **PWA 엔진 고도화**: `@ducanh2912/next-pwa` 라이브러리를 도입하여 Next.js 15 환경에서 안정적인 자산 캐싱 및 오프라인 구동 환경 구축.
- **오프라인 모드 (Offline Mode)**:
    - **네트워크 상태 감지**: `NetworkStatus` 컴포넌트를 통해 실시간 온라인/오프라인 상태를 감지하고 상단 알림 바 표시.
    - **오프라인 동작**: 인터넷 연결이 끊긴 상태에서도 기존 캐싱된 페이지를 통해 앱 구동 가능.
- **로컬 영구 저장소 (OPFS & LocalStorage)**:
    - **OPFS (Origin Private File System)**: 대용량 이미지 데이터를 브라우저 전용 로컬 파일 시스템에 바이너리 형태로 저장하여 오프라인에서도 데이터 영속성 유지.
    - **메타데이터 관리**: 자산의 정보는 `localStorage`에 저장하여 새로고침 후에도 목록 복구 가능.
- **플랫폼 맞춤 배너**:
    - **Android/데스크탑**: 설치 프롬프트 이벤트를 가로채어 직접 '설치' 버튼 제공.
    - **iOS Safari**: 공유 버튼(□↑)을 통한 수동 설치 안내 제공.
- **프로필 페이지 통합**: 프로필 페이지 내에 닫기 버튼이 없는 고정 설치 유도 섹션 추가.
- **새로고침 대응**: `localStorage`와 인라인 스크립트(`layout.tsx`)를 활용하여 페이지 새로고침 시에도 최대한 설치 상태를 유지하도록 구현.

## 2. 미해결 및 기술적 제약 사항

### A. 데스크탑/Android 'beforeinstallprompt' 이벤트의 불안정성
- **상세**: 사용자가 이미 설치 버튼을 보았으나 설치하지 않고 새로고침하거나 창을 닫은 경우, 브라우저가 자동으로 '쿨다운' 모드에 진입하여 `beforeinstallprompt` 이벤트를 한동안 발생시키지 않습니다.
- **증상**: 배너는 보이지만 버튼을 눌러도 반응이 없거나 "주소창의 설치 아이콘을 사용하라"는 메시지가 표시됨.
- **해결 방안**: 현재는 사용자에게 수동으로 주소창의 ⊕ 아이콘을 클릭하도록 안내 메시지를 표시하는 것으로 최적화했습니다.

### B. 로컬 개발 환경(IP 접속) 및 실 서비스 인증서 이슈
- **상세**: PWA 설치는 신뢰할 수 있는 HTTPS 환경에서만 작동합니다. 현재 `dev:https`는 자가 서명 인증서를 사용하므로, 브라우저가 도메인을 안전하지 않다고 판단할 경우 설치 기능을 완전히 차단할 수 있습니다.
- **증상**: 모바일에서 IP(`192.168.x.x`)로 접속 시 배너가 뜨지 않거나 설치가 작동하지 않음.
- **해결 방안**: 실 서비스 배포 시 정식 SSL 인증서를 적용하면 해결되는 문제입니다. 개발 중에는 `localhost:3000` 접속 시 가장 안정적으로 테스트 가능합니다.

### C. iOS(iPhone/iPad)의 브라우저 제약
- **상세**: iOS의 모든 브라우저(Chrome, Firefox 포함)는 내부적으로 WebKit을 사용하며, PWA 홈 화면 추가 기능은 오직 **Safari**에서만 허용됩니다.
- **증상**: iOS Chrome 등으로 접속 시 설치 안내 배너가 의도적으로 표시되지 않도록 처리함 (설치가 원천 불가능하므로).
- **해결 방안**: 사용자에게 Safari를 통해 접속하도록 유도해야 합니다.

---
> [!NOTE]
> 위 이슈들은 브라우저 엔진(V8/WebKit)의 자체 정책 및 보안 요구사항과 관련된 것이며, 웹 애플리케이션 코드 레벨에서 강제로 제어할 수 없는 부분입니다. 실 운영 환경 배포 후 정식 도메인과 인증서가 확보되면 많은 부분이 자연스럽게 해결됩니다.
