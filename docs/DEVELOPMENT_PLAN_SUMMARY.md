## Sprint 1 · Week 1–3 · 모바일 MVP 베타

**목표:** PWA만으로 베타 출시 가능한 수준

| 핵심 기능 | 비고 |
| :--- | :--- |
| 라이브러리 생성 & 온보딩 | 로컬 라이브러리 생성 (Supabase 연동은 Sprint 3) |
| 모바일 카메라 촬영 & AI 자동 태깅 | 로컬 MobileNet, 색상·구도·오브젝트 3종, 정확도 ≥ 70% |
| iOS/Android 공유 시트 저장 | 이미지·URL 로컬 저장 |
| 로컬 영구 저장 보장 | OPFS + IndexedDB, 앱 재시작 후 데이터 유지 |

**완료 기준:** 모바일 E2E (촬영 → Supabase 업로드), 오류율 < 1%

## Sprint 2 · Week 4–6 · 데스크탑 확장 & 조직화

**목표:** Electron 앱 본격 구축, 수집 채널 완성

| 핵심 기능 | 비고 |
| :--- | :--- |
| 로컬 파일 드래그&드롭 & 감시 폴더 | 8종 포맷, 드롭 후 3초 이내 썸네일 생성 |
| 브라우저 확장 7종 캡처 | Batch / Auto-Detect / Area / Element / Visible / Full Page / URL 북마크 |
| 계층형 폴더 (최대 5단계) | 14종 아이콘 × 7색 조합 |
| 폴더 자동태그 & 다중 태그 시스템 | 8개 표준 그룹 (`#color`, `#style` 등) |
| 스마트 폴더 (조건 기반) | AND/OR 복합 조건, 실시간 업데이트 |
| 별점·컬러 라벨, 메모·맥락 자동 수집 | F키 분류 단축키 |

**완료 기준:** 모바일→데스크탑 E2E 3초 이내, 태그 20,000개 환경 성능 통과

---

## Sprint 3 · Week 7–9 · 동기화 인프라 완성

**목표:** Local-First 아키텍처를 실 코드에 완전히 구현, 검색·뷰어의 안정적 토대 확보

| 핵심 기능 | 비고 |
| :--- | :--- |
| Supabase 초기화 & 인증 | DB 스키마·버킷·RLS 설정, Google/Apple OAuth, 프로필 생성 |
| Sprint 1 & 2 로컬 데이터 최초 업로드 | 기존 로컬 에셋 전체를 Supabase로 첫 동기화 |
| `Asset` 타입 동기화 필드 추가 | `syncStatus`, `updatedAt`, `remoteId` — 전체 레이어 기반 |
| `HybridAssetRepository` 완성 | `USE_MOCK` 플래그 완전 제거, 로컬 읽기/쓰기 우선 |
| Delta Sync 엔진 (`syncNow()`) | 업로드/다운로드/LWW 충돌 해결 |
| Supabase Realtime 연동 | 모바일↔데스크탑 3초 이내 메타데이터 반영 |
| 오프라인 큐 & 자동 업로드 | 네트워크 복귀 시 `pending` 에셋 자동 처리 |
| `StorageService` 용량 측정 | `navigator.storage.estimate()` 기반 3단계 감지 |
| 스마트 스토리지 최적화 | 500MB 미만 → `NVStorageOptimizeDialog` + 최적화 모드 전환 |
| 동기화 UI (배지·Sync Center·설정) | 4종 상태 배지, 글로벌 "지금 동기화" 위젯, 설정 섹션 |

**완료 기준:** 모바일→데스크탑 E2E 3초 이내, 오프라인 저장 → 복귀 후 자동 업로드 100%, `USE_MOCK` 코드 완전 제거

## Sprint 4 · Week 10–12 · 검색·뷰어 & 출시 준비

**목표:** 동기화 인프라 위에서 검색·탐색 경험 완성, 출시 품질 달성

### Week 10–11: 검색 & 뷰어

| 핵심 기능 | 비고 |
| :--- | :--- |
| 키워드 전문 검색 (고급 문법) | OR·제외·인용구, `tsvector` FTS 인덱스 |
| 색상 팔레트 검색 | HSL ±15% 허용 오차, Hex/RGB/HSL 직접 입력 |
| 복합 필터 | 포맷·날짜·별점·라벨·비율·태그 AND/OR |
| 그리드 4종 레이아웃 | Waterfall / Justified / Grid / List |
| GIF·WebP·영상 호버 미리보기 | 호버 300ms 유지 시 자동 재생 |
| 중복 파일 검사 2종 | SHA-256 동일 파일 + pHash 유사 이미지 |
| 무드보드 (레퍼런스 관계 연결) | AI 유사도 추천 최대 6개 |
| 추천 페이지 스냅샷 아카이브 | URL 저장 시 페이지 스냅샷 자동 캡처 옵션 |

### Week 12: 출시 준비

| 핵심 기능 | 비고 |
| :--- | :--- |
| 환경설정 42개 항목 | 테마·단축키·보안·동기화·썸네일 등 |
| 집중 QA | 전체 US 인수 조건 100% · 4개 브라우저 · 2개 OS |
| 배포 패키지 | macOS `.dmg` + Windows `.exe` NSIS |

**완료 기준:** 10만 에셋 검색 200ms 이내, 레이아웃 4종 정상 전환, 크래시율 < 0.5%, 심각 버그 0건
